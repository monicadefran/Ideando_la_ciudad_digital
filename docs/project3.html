Tutorial 3: Cash Register

  1. Home
  2. Tutorial 3: Cash Register

Objetivo:

El objetivo de este ejemplo es mostrar cómo se importar un proyecto desde el ordenador, cómo se leen código QR o código de barra.

Pasos a Seguir

  1. Abrir el entorno de desarrollo App Inventor.
    1. http://appinventor.mit.edu/explore/#
    2. Pulsar el botón create app!
    3. Selecciona una cuenta de Google
  2. Pulsa el menú Projects -> Import project (.aia) from my computer... (Descargar el proyecto que se proporciona en el botón)
  3. Cambia a la pestaña Blocks (Arriba a la derecha)
    Pestaña para cambiar a modo desarrollador de lógica.
  4. Lo primero de todo se creará una lista vacia que irá almacenando los precios que se lean
    Creación de una lista vacia
  5. Tras ello se indicará que tras pulsar el botón se encenderá el modo leer código de barras o código QR
    Inicio de lectura de Código QR o Código de Barras tras pulsar el botón
  6. En la última parte de la lógica se indicará que tras leer un código QR o de barras si el valor es numérico entonces se generá un valor aleatorio de precio, se hará un sonido de caja registradora, se añadirá el valor generado aleatoriamente a la variable que contiene la lista de precios, se mostrará la lista de precios para finalmente calcular el total. Si el valor leido no es un número se mostrará un mensaje de error.
    Lógica tras leer el código de barras o código QR
  7. Pulsa el botón del título Build y seleciona la opción App (provide QR code for .apk).
    Pestaña para compilar y descargar la aplicación.
  8. Abre la aplicación App Inventor desde el móvil y escanea el código QR.
    Arrancar App Inventor.
    Pulsar Escanear Código QR.
  9. Instala la aplicación y pruébala.
    Aceptar Instalar aplicación
    Aceptar Instalar aplicación sin firma reconocida
