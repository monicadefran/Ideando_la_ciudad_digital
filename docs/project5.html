Tutorial 5: Talk to me (con acelerómetro)

  1. Home
  2. Tutorial 5:Talk to me (con acelerómetro)

Objetivo:

El objetivo de este tutorial es hacer una aplicación donde podamos introduccir texto y escucharlo. También haremos que cada vez que agitamos el teléfono se dispara un evento que en este caso es un texto predeterminado

Pasos a Seguir

  1. Abrir el entorno de desarrollo App Inventor.
    1. http://appinventor.mit.edu/explore/#
    2. Pulsar el botón create app!
    3. Selecciona una cuenta de Google
  2. Pulsa el menú Projects -> Star new project
  3. Lo primero que vamos hacer es crear un campo de texto y después vamos a añadir un botón
  4. Seguidamente vamos a añadir el elemento TextToSpeech que vamos a encontrar en la pestaña de media. Este elemento se posicionará como un componente no visble
  5. Ahora vamos a implementar la lógica de nuetra app. Para ello lo que vamos a cambiar a la parte de Blocks.
  6. Primero vamos a diseñar la lógica del botón
    1. Incluimos la caja del botón que nos permite generar una interacción cuando lo pulsamos
    2. Después tenemos que añadir el llamamiento al texto para que lo hable
    3. Por último tenemos que añadir como input el texto añadido por el usuario, como podemos ver en la imágen siguiente
  7. Seguidamente vamos a programar la acción del acelerómetro, para ello hay que añadirlo como elemento primero
  8. Vamos a generar la lógica del acelerómetro.
    1. Añadimos el bloque del aclelerómetro que actuará cuando agitamos el móvil
    2. Después tenemos que añadir el llamamiento al texto que va a sonar cuando movamos el móvil
    3. Por último tenemos que añadir el texto que queremos que suene
