Objetivo:

El objetivo de este tutorial es mostrar como se comportan ciertos elementos de appInventor y aprender el funcionamiento del bloque if else

Pasos a Seguir

  1. Abrir el entorno de desarrollo App Inventor.
    1. http://appinventor.mit.edu/explore/#
    2. Pulsar el botón create app!
    3. Selecciona una cuenta de Google
  2. Pulsa el menú Projects -> Import project (.aia) from my computer... (Descargar el proyecto que se proporciona en el botón)
  3. Añadir las tres imágenes del galería de imágenes que vamos a crear...(Descargar el proyecto que se proporciona en el botón)
  4. Lo primero que hacemos es decargar el siguiente archivo.
  5. Descomprime el archivo .zip y añade las imagenes al aparado de recursos media como te mostramos acontinuación
  6. Ve al apartado Media, situado abajo a la derecha y presionar el boton Upload File y cargar las imágenes como se muestra a continuación
