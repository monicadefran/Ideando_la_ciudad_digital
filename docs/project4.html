Objetivo:

El objetivo de este tutorial es hacer una galería de imágenes que funcionen con un botón de next. Dentro de ese bloque vamos a aprender como se comportan ciertos elementos de appInventor y el funcionamiento del bloque if else

Pasos a Seguir

  1. Abrir el entorno de desarrollo App Inventor.
    1. http://appinventor.mit.edu/explore/#
    2. Pulsar el botón create app!
    3. Selecciona una cuenta de Google
  2. Pulsa el menú Projects -> Import project (.aia) from my computer... (Descargar el proyecto que se proporciona en el botón)
  3. Añadir las tres imágenes del galería de imágenes que vamos a crear...(Descargar el proyecto que se proporciona en el botón)
  4. Lo primero que hacemos es decargar el siguiente archivo.
  5. Descomprime el archivo .zip y añade las imagenes al aparado de recursos media como te mostramos acontinuación
    • Ve al apartado Media, situado abajo a la derecha, y presionar el bóton Upload File para cargar las imágenes como se muestra a continuación
  6. Añadimos un elemento botón y un elemento imágen
  7. En la parte de propiedades de cada elemento, primero cambiamos las propiedades de la imagen, Height: Fill partent... y Width: Fill partent... En Picture eleguimos las tres imágenes (img1.jpg, img2.jpg, img3.jpg) y nos aseguramos de que el botón de check de ScalePictureToFit esté selccionado. Cambiamos el nombre del botón a Nextboton, con los botónes que encontramos en Components
  8. Nos vamos a la parte de desarrollo para implementar toda la lógica de la aplicación
    1. Los primero que vamos hacer es crear una variable global, que encontramos en el bloque de Variables. Y lo vamos a unir con una lista donde vamos a integrar las tres imágnes que tenemos en media, como se muestra en la imágen de abajo.
      • Por defecto algunos elementos de appInventor vienen con ciertas propiedades que podemos cambiar. En el caso de la lista vamos a añadir un Item más para poder tener nuetras tres imagenes. Para ello pinchamos en la rudeda dentada de la izquierda del elemento y arrastrando añadimos el item.
  9. Ahora vamos a darle un nombre a la variable global: imageFiles. El primer modulo de programación nos quedaría como acontinuación.
  10. El siguiente paso es iniciar otra varaible global que nos va a ayudar a recorrer la lista. Esta varaible la vamos a llamar "index", y la vamos a iniciar en 1. Deberíamos tener algo parecido a la imagen que mostramos a continuación.
  11. Por útimo vamos a implementar la lógica del botón. El funcionamiento del botón responde a que cada vez que hacemos click pasa a la siguiente imágen.
    1. Lo primero es elegir uno de los elementos de control del botón y que cuando pulsemos el botón ocurra algo.
    2. Después debemos inroduccir dentro del comportamiento del botón la lógica del if.
    3. La condición dentro del if la ponemos de la siguiene manera, si la variable global index es menor que la longuitud de la lista de la variable global de imágenes, le sumamos 1 a la variable global index, sino lo que hacemos es que la variable global index sea 1, como al inicio. Como se muestra acontinuación
  12. En el útimo bloque lo que hacemos es decirle donde tiene que ir a buscar y coger las imágenes.
  13. A continuación explicamos de donde se coge cada bloque. Estos se corresponden con los colores.
